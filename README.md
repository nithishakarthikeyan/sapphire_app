# Sapphire
*A Flutter-based home service booking application*

## Overview
This is a cross-platform mobile application built with **Flutter** that allows users to browse, book, and manage household services such as cleaning, painting, plumbing, electrical work, and more.

The app demonstrates a complete service-booking workflow — from service discovery to booking confirmation — with a focus on UI design, navigation, and user interaction.

## Features
- Service Discovery
- Service Providers
- Booking Flow
- Booking Summary & Confirmation
- User Dashboard
- Navigation

## Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **UI:** Material Design
- **Navigation:** Navigator (named routes & `MaterialPageRoute`)
- **State Management:** Stateful Widgets
- **Third-Party Packages:**
  - `numberpicker`
